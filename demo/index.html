Tag / Token Input

Lightweight (1.6kb gzipped) jQuery plugin with optional autosuggest support.

Full Readme

  • See GitHub.

Options

  • source – An array or method providing a list of words, if you want to use the autosuggest feature.
  • allowUnknownTags – If autosuggest is enabled, prevents user from creating tags not on the list.
  • numToSuggest – Max number of items to show in autosuggest list.
  • namespace – Customized prefix for all generated HTML, for your CSS-styling pleasure.
  • label – Title of the input.
  • placeholder – Input placeholder text.
  • separators – Break tokens on any characters you like. Include ' ' to prevent multiple words per tag.
  • callback – Function to call when token list is updated.
  • xContent – Content of the 'delete tag' button.
  • onclick – Function to call when a token is clicked. Token text is given as a parameter.
